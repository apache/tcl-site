Rivet

Webscripting for Tcl'ers

Download Apache Rivet

Source Code

The latest stable Rivet version is 3.2.2 and it may be downloaded from one the links pointing to ASF mirror sites

http://www.apache.org/dyn/closer.lua/tcl/rivet

SuSE

Thanks to the efforts of Harald Oehlmann, Apache Rivet is available from OpenSUSE for many flavours of Linux

RPM

Jeff Lawson prepared a .spec file for RPMforge that can be used to build an .rpm out of the source code. Installation instructions are available at Repoforge.

Debian and Ubuntu

Rivet was kindly sponsored by Sven Hoexter, Debian Developer. You can download it from from Debian as libapache2-mod-rivet .

Rivet is also available on launchpad.net for the Ubuntu series of Linux

FreeBSD

Peter Lai is maintaining a FreeBSD port available as www/mod_rivet

Development code on GitBox or GitHub

Apache Rivet is actively developed - you can always get the full git repository from either GitBox.

git clone https://gitbox.apache.org/repos/asf/tcl-rivet.git rivet

or from GitHub

git clone https://github.com/apache/tcl-rivet.git rivet
