Rivet 2.0 support discontinued

Rivet 2.0 support discontinued

The Rivet team voted for interrupting the support to Rivet 2.0, as version 2.1 extends 2.0 and it's fully compatible. Furthermore, the main reason for keeping Rivet 2.0 was providing a mod_rivet version able to build and run with Apache 1.3. Since this version of the Apache HTTP web server was deprecated this motivation has now dropped.

Rivet version 2.0.6 artefact will be removed from the ASF mirrors but will continue to be available from the ASF Archives
