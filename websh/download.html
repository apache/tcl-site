If you would like to be notified when new versions of Websh become available for download, you can subscribe to the Websh user mailing list.

To access the latest source from the Subversion repository, use the following (read-only) URL:

    http://svn.apache.org/repos/asf/tcl/websh/trunk
    Overview of changes in trunk since last release

A typical command to checkout the code would be:

    svn co http://svn.apache.org/repos/asf/tcl/websh/trunk websh

Note: Websh 3.6.0b5 is known to work with Sun Solaris 8, 9, and 10, Linux RedHat 8, CenOS 4.6, Suse9.1, Windows XP and Vista, Mac OSX 8.2 und 8.3. Other Unix platforms are likely to work too.
