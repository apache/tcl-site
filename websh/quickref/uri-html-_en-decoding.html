Uri-/html- en-/decoding

web::htmlify

web::htmlify ?-options? text.

Options: -numeric.

Return HTML-compliant text with HTML encoded entities in mnemonic form (e.g. &auml;) or in numeric form (e.g. &#228;) if the option -numeric is specified.

web::dehtmlify

web::dehtmlify text

Remove all HTML-tags from text and translates all HTML entities to their corresponding ISO-8859-1 characters.

web::uriencode

web::uriencode text

Return URI-compliant text.

web::uridecode

web::uridecode text

Decode URI-compliant text.

Example 12. web::htmlify

	  % web::htmlify <
	  &lt;
	  % web::htmlify -numeric <
	  &#60;
	  % web::dehtmlify "&lt; &#60;"
	  < <
	  % web::uriencode "Hello, world!"
	  Hello%2c+world%21
	  % web::uridecode "Hello%2c+world%21"
	  Hello, world!
	  %
